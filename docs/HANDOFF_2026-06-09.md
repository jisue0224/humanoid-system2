- `vx=1.0` moves forward.

### Step 3

- Static obstacle at `(2.5, 0.0, 0.5)`, size `(1.0, 1.0, 1.0)`.
- Policy-only direct goal tracking gets blocked near obstacle front.
- Progress uncertainty separates no-obstacle vs obstacle cases.
- Threshold reused later: `0.0247`.

Uncertainty formula:

```python
progress = distance[t - 5] - distance[t]
uncertainty = max(0.0, 0.02 - progress)
```

Trigger: `uncertainty >= 0.0247`.

### Step 5b Waypoint LLM

- Script: `scripts/llm_waypoint_navigation_h1.py`
- Merge script: `scripts/merge_llm_waypoint_navigation_results.py`
- LLM model: `gpt-5.4`
- LLM output JSON:

```json
{
  "reasoning": "one sentence",
  "waypoint_x": 1.7,
  "waypoint_y": -1.1
}
```

- H1 tracks the waypoint with the Step 2 goal controller.
- Waypoint radius: `0.8 m`.
- Waypoint timeout: `320 steps`.
- Max episode length: `700 steps`.
- Episodes: `20`.

Results:

| Method | success_rate | mean_len | mean_final_distance | LLM calls |
|---|---:|---:|---:|---:|
| policy_only | 0.00 | 700.0 | 3.095 m | 0 |
| pure LLM waypoint | 0.40 | 639.7 | 1.646 m | 112 |
| LLM waypoint + geometric postprocess | 1.00 | 509.3 | 0.338 m | 60 |

Key interpretation:

- System 2 should output a waypoint/subgoal, not direct motor-ish velocity commands.
- A small geometric postprocess makes the LLM waypoint physically usable for the H1 forward-walking controller.

Postprocess rules:

- If H1 is still close to the obstacle front and lateral clearance is insufficient, force a first-stage lateral escape waypoint beside/slightly behind robot.
- If H1 already has lateral clearance, keep the same side and force the waypoint past the obstacle.
- If waypoint is inside obstacle, move it outside.

## Useful Commands

Clone/pull fresh repo:

```bash
cd /workspace
git clone https://github.com/jisue0224/humanoid-system2.git
cd humanoid-system2
```

Check latest commit:

```bash
git log -1 --oneline --decorate
git status --short
```

Run waypoint postprocessed experiment:

```bash
source env_isaaclab/bin/activate
OMNI_KIT_ACCEPT_EULA=YES PYTHONUNBUFFERED=1 TERM=xterm \
  external/IsaacLab/isaaclab.sh -p scripts/llm_waypoint_navigation_h1.py \
  --headless --device cuda:0 \
  --condition uncertainty_switching \
  --num_envs 20 --episodes 20 --max_steps 700 \
  --task Isaac-Velocity-Flat-H1-v0 \
  --output_dir experiments/llm_waypoint_navigation_postprocessed
```

Disable waypoint postprocess for pure LLM waypoint:

```bash
OMNI_KIT_ACCEPT_EULA=YES PYTHONUNBUFFERED=1 TERM=xterm \
  external/IsaacLab/isaaclab.sh -p scripts/llm_waypoint_navigation_h1.py \
  --headless --device cuda:0 \
  --condition uncertainty_switching \
  --disable_waypoint_postprocess \
  --num_envs 20 --episodes 20 --max_steps 700 \
  --task Isaac-Velocity-Flat-H1-v0 \
  --output_dir experiments/llm_waypoint_navigation
```

Merge waypoint results:

```bash
python scripts/merge_llm_waypoint_navigation_results.py \
  --output_dir experiments/llm_waypoint_navigation_postprocessed
```
